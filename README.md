# Ecommerce QA Project

## Project Overview
This project demonstrates **end-to-end QA testing** of a demo e-commerce site (https://www.saucedemo.com/)  using:

- Manual Testing (test cases, bug reports)
- Selenium Automation (Python + Pytest)
- Database Validation (MySQL/PostgreSQL)

## Folder Structure
- `tests/` - Selenium test scripts
- `pages/` - Page Object Model files
- `db/` - Database scripts
